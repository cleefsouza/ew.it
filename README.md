# ew.it

> Práticas do curso de Node.js  do Erick Wendel

### Requisitos
- Node.js
- NPM

### Executando
- Clone o projeto:
  ```shell
  git@github.com:cleefsouza/ew.it.git
  ```

- Acesse a pasta de seu interesse e execute os seguinte comandos:
  ```shell
  # Instalar as dependencias
  
  npm install

  # Executar o arquivo main
  
  node index.js
  ```
